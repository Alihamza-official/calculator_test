# What is this ?

Calculator

# Installation
`npm i helloocalculator`

## Operations
* add 
* subtract
* multiply
* divide

const {calculator} = require('./utility');

console.log(calculator.add(2, 4)); // 6
console.log(calculator.subtract(6, 4)); // 2
console.log(calculator.multiply(2, 4)); // 8
console.log(calculator.divide(8, 4)); // 2
